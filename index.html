Battery Passport Demo

Static Battery DPP (Digital Product Passport)

Public, machine-readable passport pages resolved from a QR-linked UID using only static files (CSV index + JSON passports).

Find a Battery Passport

Search by UID, GTIN, serial number, model, or category.

Demo Notes

  • Static hosting only: no backend, no database.
  • UID routing supports a GS1 Digital Link style path (`/p/01/.../21/...`).
  • Passport data is stored in JSON and indexed via `data/products.csv`.
  • Public vs restricted access is a UI concept only in this demo.
