電池護照示範 / Battery Passport Demo

靜態電池 DPP（數位產品護照）/ Static Battery DPP (Digital Product Passport)

使用純靜態檔案（CSV 索引 + JSON 護照）透過 QR 綁定 UID 解析公開電池護照頁面。
Public, machine-readable passport pages resolved from a QR-linked UID using only static files (CSV index + JSON passports).

查找電池護照 / Find a Battery Passport

可用 UID、GTIN、序號、型號或類別搜尋。 / Search by UID, GTIN, serial number, model, or category.

示範說明 / Demo Notes

  • 純靜態託管：無後端、無資料庫。 / Static hosting only: no backend, no database.
  • UID 路由支援 GS1 Digital Link 風格路徑（`/p/01/.../21/...`）。 / UID routing supports a GS1 Digital Link style path (`/p/01/.../21/...`).
  • 護照資料以 JSON 儲存，並由 `data/products.csv` 建立索引。 / Passport data is stored in JSON and indexed via `data/products.csv`.
  • 公開/受限存取僅為 UI 概念展示，未實作真實驗證。 / Public vs restricted access is a UI concept only in this demo.
